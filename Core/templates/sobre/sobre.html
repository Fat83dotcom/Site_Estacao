O que é a Estação?

A estação meteorológica BrainStorm tecnologia é um projeto baseado em Arduíno, desenvolvido pela mente criativa de Fernando Mendes, estudante em Análise e Desenvolvimento de Sistemas. Esse projeto de baixo custo e de código aberto tem como objetivo monitorar eventos climáticos em tempo real, fornecendo dados precisos para análises do comportamento do clima em uma determinada região.

O Arduíno

A estação utiliza diversos sensores, como o sensor BME280, capaz de captar temperatura, umidade e pressão em um mesmo chip, e o termistor NTC 10k, um resistor que varia sua resistência elétrica de acordo com a temperatura ambiente. Esses sensores são conectados ao Arduíno Nano e montados em uma protoboard. Além disso, a estação conta com um visor OLED 128x64 para visualização dos dados.

A Interface Gráfica

A interface gráfica do programa é simples e intuitiva, permitindo ao administrador da estação configurar o envio de e-mails periódicos com os dados atualizados. Além disso, o programa oferece um painel que exibe em tempo real o andamento do envio dos e-mails e os dados captados.

A versão atual, que pode ser encontrada aqui , é estável e ocupa pouca memória, mesmo armazenando um cache de dados, conforme o tempo definido para os e-mails. Os e-mails contem informações entre os períodos de médias, máximas e mínimas com gráficos anexos em pdf mostrando segundo a segundo os registros.

A Captação

A captação dos dados é realizada de forma contínua, com os sensores enviando informações para o Arduíno através de uma conexão serial. O Arduíno, por sua vez, responde às solicitações do programa em Python, que atua como cliente, buscando os dados atualizados a cada segundo. Esses dados são então gravados em arquivos .csv, organizados por mês/ano, no servidor local.

Diariamente, à meia-noite, um algoritmo é acionado para tratar os dados do dia anterior. Esse algoritmo realiza análises estatísticas básicas, como média, máxima, mínima, mediana e moda, e salva um resumo dessas informações em um banco de dados localizado em um servidor remoto.

A Exibição

Para visualizar os dados armazenados, o projeto utiliza um aplicativo WEB desenvolvido com o framework Django. O aplicativo oferece duas formas de exibição dos dados: através de tabelas, onde é possível aplicar filtros para facilitar a visualização, e através de gráficos, que são uma forma eficiente de analisar as informações. Essa interface amigável proporciona uma experiência intuitiva ao usuário.
